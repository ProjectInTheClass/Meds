# ⏰ 자취생들을 위한 재료관리 어플 , Meds(Management Expiration Dates)

### 개발환경
- IDE : Xcode
- Language : Swift

한양대학교 소프트웨어 스튜디오 수업의 일환으로 만들어진 IOS APP 입니다.

음식을 사놓고 유통기한을 몰라 버리는 경우가 많으신가요? 걱정하지 않으셔도 됩니다. 이젠 Meds가 도와드리겠습니다.

주요 기능

- 바코드를 통해 재료의 유통기한을 관리

- 유통기한 3일전 알림.

- 재료에 대한 간단한 음식 추천.
