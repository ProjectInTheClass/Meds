앱 소개

Meds는 자취생들을 위한 유통기한 관리 앱입니다.
혼자 살면 음식을 해먹고 남은 재료를 버리는 경우가 부지기수 입니다. 이런 아이디어 착안해 자취생들에게 신선한 제품을 소비 할 수 있는 문화를 정착시킴으로써 건강한 식단을 만들어먹는 계기가 될 것입니다.
또한 음식물 쓰레기를 줄임으로써 지구온난화를 막는데에 큰 효과를 낼 것으로 기대합니다.

주요 기능

  • 재료 관리

    직접 핸드폰을 이용하여 저장한 재료로 실제 유통기한을 관리해주는 모습입니다.

  • 바코드

    바코드의 종류가 여러가지 있는데 90%이상 인식할 수 있도록 하였습니다. 바코드를 인식하면 초록색 네모칸이 뜨면서 인식이 되었다는 창이 뜨게 됩니다.

  • 제품 직접입력

    바코드를 통해 입력하지 못하는 경우를 대비하여 제품 직접 입력 칸을 만들어서 재료를 관리 할 수 있도록 만들었습니다.

주요 기능

  • 메뉴 추천

    냉장고 속 재료로 어떤 요리를 해먹을까 고민하는 분들을 위해 추천기능으로 유튜브와 연동하도록 만들었습니다.

  • HealthKit 연동

    이것은 사용자의 데이터를 불러와 직접 BMI를 계산할 결과입니다. 사용자는 몸무게를 입력하면 그 정보를 BMI로 바꿔 건강 상태를 알고 헬스킷에 저장하도록 하였습니다.

  • 알람 기능

    저희는 유통기한 정보를 구현할 수 있도록 하였습니다. 이에 따라 3일전에 핸드폰과 애플와치에 알람이 올 수 있도록 하였습니다.
